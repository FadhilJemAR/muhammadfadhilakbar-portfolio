Tentang saya

Halo! Saya adalah seorang Fullstack developer pemula yang punya antusiasme tinggi dalam membangun tampilan web yang bersih dan responsif. Berawal dari rasa penasaran gimana sebuah website bekerja, saya mulai mendalami HTML, CSS, dan JavaScript secara otodidak. Saya sangat menikmati proses mengubah baris kode menjadi desain visual yang interaktif. Saat ini, saya fokus membangun proyek-proyek personal untuk mengasah logika dan estetika desain saya, sebenarnya saja juga mempunya beberapa proyek fullstack yang dibangun dengan runtime nodejs, namun tak hanya nodejs saya sudah memahami beberapa fundamental ReactJS, MongoDB, dan library ExpressJS yang mendukung pembuatan server di NodeJS
